Não. A plataforma foi criada para ser simples e intuitiva. Se você sabe usar o WhatsApp, você usa a MS Color IA.

Sim. A plataforma foi feita exatamente para isso. Atenda de qualquer lugar, pelo celular, em qualquer país.
